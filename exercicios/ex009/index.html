RESUMO de TAGS

    
    

        <ol> = ordered lists >>Para lista com necessidade de ordenamento. Necessário abertura e fechamento de tag.

        <li> = list item >>Para adição de item na lista. Necessário abertura porem o fechamento da tag não é obrigatório.

        <ul> = unordered lists >>Para lista sem necessidade de ordenamento. Necessário abertura e fechamento de tag.



        OBS: Para ol(ordered lists) podemos incluir parâmetros (type) para escolhermos os marcadores da lista. 

        
    1 - Valor padrão. Cria listas numeradas. EX: 1, 2, 3, 4, ...

        
    A - Cria listas alfabéticas em maiúsculas. EX: A, B, C, D, ...

        
    a - Cria listas alfabéticas em minúsculas. EX: a, b, c, d, ...

        
    I - Cria listas com algarismos romanos maiúsculas. EX: I, II, III, IV, ...


        
    i - Cria listas com algarismos romanos minúsculas. EX: i, ii, iii, iv, ...


        Podemos também informar o start="5" para definir o ponto de partida da lista. Definição de start deve ser sempre com o número correspodente com a ordem, para casos de letra, numero correspondente à letra solicitada.
